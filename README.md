# Finance Dashboard

A simple  and interactive financial tracking dashboard to view income, expenses, and balance patterns.

## Technologies used
- HTML5, CSS3, JavaScript
- Bootstrap 5
- Chart.js

## How to Run
1. Download the project
2. Open `index.html` in browser

## Features
- Income, Expense, and Balance summary cards
- Category-wise spending chart(Pie Chart)
- Monthly breakdown (Bar Chart)
- Filter data and view updated totals
- Add transactions (Admin only)
- Role based UI — Admin and Viewer
- Dark mode toggle
